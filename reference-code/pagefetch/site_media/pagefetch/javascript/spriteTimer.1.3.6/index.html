Example A

This timer will count down until it reaches 0 and then run the function runWhenFinished.

			
$('.exampleA .timer').spriteTimer({
   'seconds': 37000,
   'isCountDown': true,
   'digitImagePath': 'numbers.png',

});

Example C

Start Me Reset

This timer has a defined start/stop button as well as a reset. The startTimerCallback and stopTimerCallback functions have also been set to toggle the start button's text.

			
function toggleStartBtn() {
   var btn = $('#startBtn');
   if (btn.text() == 'Start Me') {
      btn.text('Stop Me');
   } else {
      btn.text('Start Me');
   }
}
				
$('.exampleC .timer').spriteTimer({
   'digitImagePath': 'numbers.png',
   'startButton': '#startBtn',
   'resetButton': '#resetBtn',
   'startTimerCallback': toggleStartBtn,
   'stopTimerCallback': toggleStartBtn,
   'resetTimerCallback': toggleStartBtn,
   'isCountDown': false
});

Example D

This timer will count up indefinitely and show the value in seconds here:

Mouse over this box to start the timer.
Mouse over this box to stop the timer.
Mouse over this box to reset the timer.
			
$('.exampleD .timer').spriteTimer({
   'digitImagePath': 'numbers.png',
   everySecondCallback: function(sec) {
      $('#seconds').val(sec);
   },
   'isCountDown': false
});

$('#stopMe').bind('mouseover', function() {
   $('.exampleD .timer').trigger('stopTimer');
});
$('#startMe').bind('mouseover', function() {
   $('.exampleD .timer').trigger('startTimer');
});
$('#resetMe').bind('mouseover', function() {
   $('.exampleD .timer').trigger('resetTimer');
});
