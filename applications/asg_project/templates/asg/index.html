Abstract Search Games

Test how good you are at searching by playing some of our abstract search games.

  • In each game you will have two options: query or assess.
    • query: will bring back a set of new results
    • assess: will open up the new unopened document, and you recieve the number of points associated with that document.
  • Because this is an abstracted search game. you dont need to type in queries, nor judge documents for relevance.
  • Your goal is to try and find as much information (denoted by total points)
  • However, each time you issue a query or assess a document it will take time
  • At the start of each game you are given a certain amount of time which you can spend on querying or assessing.
  • Different games will put you in different conditions, different amounts of time, different costs, different rewards.
  • Put your search skills to the test.
  • Register and login to Play
