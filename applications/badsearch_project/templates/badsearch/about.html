About Badsearch


Badsearch is a search application developed by Iain Hendry, an MSc IT student at the University of Glasgow, as part of a research project.

The application is powered by Microsoft Bing, and is intended for to be used in place of your favourite search engine for the duration of the experiment. If you would like to participate, please create an account below to begin searching.

If you have any questions or problems regarding Badsearch please contact Iain at: 0900282h@student.glasgow.ac.uk or the project supervisor, Dr. Leif Azzopardi, at: Leif.Azzopardi@glasgow.ac.uk
